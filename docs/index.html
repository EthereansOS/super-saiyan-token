SuperSaiyanToken

Super Saiya-jin Token is a ethItem Token Standard which includes capabilities to be used as a Voting Tokens for DFO. It is linked to a specific DFO through the Double Proxy, and every function that writes on the storage can be just triggered through a Proposal by the Token Holders of the linked DFO.
